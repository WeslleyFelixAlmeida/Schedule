**Schedules**

<br>
Sistema de criação e gerenciamento de eventos desenvolvido com TypeScript, React e Vite. Uma aplicação web moderna e responsiva que permite organizar compromissos, agendar atividades e visualizar sua agenda de forma intuitiva. Ideal para quem busca uma solução prática para controle de eventos pessoais ou profissionais.
<br>

🎯 Sobre
Aplicação web para criar e gerenciar eventos de forma simples e eficiente.
